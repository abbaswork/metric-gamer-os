# Metric: Map Design
**Genre:** MOBA
**Sub-genre:** Hero Brawler
**Type:** Qualitative (community consensus)
**Direction:** Delight — 5 = varied, well-designed maps that meaningfully change how each mode plays and which brawlers/heroes are optimal, 1 = interchangeable maps that don't affect strategy or character selection

## Why This Matters

Map design is the primary variety engine in hero brawlers — more so than in classic MOBAs, where the single primary map is the competitive backbone and variety comes from the meta and champion pool. Hero brawlers substitute map and mode variety for the deep itemisation layer they remove: the reason a Brawl Stars session feels different across an evening of play is that each mode runs on maps with different geometry, choke points, wall placements, and objectives. HotS specifically built its entire strategic identity around this: "with over a dozen maps, many of which have alternative versions in other Heroes of the Storm game modes like Brawl Mode, it is rare to play on the same Battleground in the same session."

The research identifies map interaction as an explicit MOBA appeal driver: "MOBAs tend to design maps in ways that allow classes and champions to have interesting interaction with the environment." In hero brawlers this interaction is more direct than in classic MOBAs — terrain cover, wall geometry, and objective placement directly determine which brawlers are viable (close-range brawlers dominate tight-corridor Brawl Stars maps; long-range brawlers struggle there). Map design quality in this sub-genre directly determines both the replayability of individual modes and the strategic depth of brawler selection.

## Rubric

| Score | Description | What This Means for You |
|---|---|---|
| 5 | Large pool of maps with meaningfully different geometry — wall placements, choke points, open areas, and objective positions that change both the strategy and the optimal brawler/hero picks noticeably between maps. Playing on different maps within the same mode feels like a different game, not a reskin. Objectives on each map create predictable conflict moments that produce exciting turning points. Map rotation or variety ensures you rarely play the same configuration twice in a session. | Which map you're on is a genuine strategic consideration. The game feels different map to map. |
| 4 | Strong map variety with clear strategic differentiation between most maps. Some maps are stronger or weaker for specific character types. Objectives create reliable conflict moments. A few maps feel less distinct or are rarely favoured, but the overall pool is strong. | Good map variety. You notice which map you're on and it influences how you play. |
| 3 | Adequate maps but limited geometric variety — many maps feel like minor variants of each other with slightly different wall placements. Objective positions are present but don't reliably create exciting turning points. Brawler/hero advantage on specific maps is minimal enough that character selection isn't meaningfully map-influenced. | Maps serve the game without adding much to it. You stop noticing which map you're on fairly quickly. |
| 2 | Few distinct maps with minimal strategic differentiation. Most maps feel interchangeable. Objectives don't reliably create conflict moments — they resolve too quickly or too predictably. Brawler/hero advantage is map-agnostic. | Map select is cosmetic. It changes the background but not how you play or who you pick. |
| 1 | A single map or maps so similar they function identically. Map design doesn't function as a strategic system — it's an arena. Objectives are absent or irrelevant. | The map is a backdrop, not a game system. Strategy is identical regardless of where you're playing. |
