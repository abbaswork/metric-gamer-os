# Metric: Mode & Map Variety
**Genre:** MOBA
**Sub-genre:** Hero Brawler
**Type:** Qualitative (community consensus)
**Direction:** Delight — 5 = large selection of distinct game modes with different objectives and win conditions that sustain long-term engagement, 1 = single mode with no meaningful variety in how the game is won

## Why This Matters

Mode variety is the depth substitute in hero brawlers — it is what compensates for the removal of itemisation and the simplified mechanics that define the sub-genre. Naavik identifies this directly: "the radically different Events expedite the use of various Brawlers because different character types excel in distinct modes." Without mode variety, a hero brawler is a single game with simplified mechanics and no build system — the combination produces very little to sustain long-term engagement. With mode variety, the same roster of brawlers and heroes can feel like multiple different games: a gem collection mode plays differently than a siege mode which plays differently than a battle royale mode.

The strategic layering this creates is the closest substitute hero brawlers have for classic MOBA build depth: instead of "what do I buy against this enemy team?" the question is "which brawler is optimal for this mode on this map?" Naavik notes that "short rounds and near-instant matchmaking keep players hooked" — the mode-to-mode rotation amplifies this by ensuring each short session feels different from the last. HotS built its entire identity around map-based mode variety: "with over a dozen maps, many of which have alternative versions in other Heroes of the Storm game modes like Brawl Mode, it is rare to play on the same Battleground in the same session." Brawl Stars at its best offers 20+ distinct game modes, each with its own map pool and strategic demands.

## Rubric

| Score | Description | What This Means for You |
|---|---|---|
| 5 | Large selection of distinct game modes — at least 5 modes with meaningfully different objectives, win conditions, and optimal brawler/hero picks. What works in one mode doesn't simply transfer to another. New modes are added regularly and feel genuinely distinct rather than minor variations on existing ones. Playing through a full session of different modes feels like playing multiple different games rather than rotating through skins of the same experience. The mode selection actively encourages exploring the full roster. | Every mode feels like a different game. Sessions stay fresh because the goal changes, not just the map. |
| 4 | Strong mode variety with 3–5 distinct modes. Each has a different objective structure and favours different playstyles. New modes are added periodically. A few modes feel similar to each other or have clearly superior versions that the player base converges on, but the overall selection is genuinely diverse. | Good mode variety. Each mode requires a different approach and there are enough to rotate through without sessions feeling repetitive. |
| 3 | 2–3 modes that are distinct in objective but similar enough in execution that the same brawlers/heroes and strategies tend to perform well across them. New modes are rare or feel like reskins. Sessions can start to feel repetitive after extended play even when rotating through available modes. | Adequate variety at the start but modes stop feeling different from each other within a few sessions. |
| 2 | 1–2 modes with limited variation between them. The same brawlers and strategies dominate regardless of mode. Mode selection feels like choosing a different UI skin over the same underlying game. | Modes are superficially different but play identically. The variety is cosmetic. |
| 1 | Single mode or multiple modes that are indistinguishable in execution. No meaningful variation in objectives, optimal picks, or strategy. The concept of mode variety is absent — you're playing the same game every session. | There is no mode variety. Every session is the same. |
