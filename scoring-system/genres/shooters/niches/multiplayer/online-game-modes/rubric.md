# Metric: Game Modes
**Sub-genre:** Shooters — Multiplayer (Mode Variety & Design)
**Type:** Qualitative (community consensus)
**Direction:** Delight — 5 = every mode is genuinely distinct, complements the maps and class system, and is playable offline as well as online, 1 = modes are thin, forgettable variations with no offline fallback

## Why This Matters

A game's mode list only matters if each mode is actually worth playing. Good mode design gives every mode a distinct identity that complements the maps and class system around it rather than feeling like the same firefight with a different win condition — Battlefield's Conquest and Rush are the reference point here, each restructuring the same maps around a different flow: Conquest spreads objectives out so the whole map sees play, while Rush funnels both teams down a linear attack/defend lane that creates a dynamic, shifting front line as sectors fall. Call of Duty: Black Ops 2 is the community's reference for variety done right in a different direction — Gun Game and Sticks and Stones are still cited as fan favorites years later because they're genuinely different games built on the same core shooting, not a ruleset swap.

Whether a mode is playable offline is a second, distinct dimension of mode design, not just a nice-to-have — this is the OP's own wording from the r/fps intro post backing this research: "the game modes are actually active, I can run them both online and offline." Call of Duty's Black Ops games are the community reference for doing this right: every mode and map is playable offline against bots via split-screen, with full access to loadouts and no leveling gate, giving players a way to experience every mode regardless of who else is online. The failure case is increasingly common in newer online-first shooters — titles like Valorant have no true offline mode at all, not even for practice, which is a well-documented player frustration and stands in direct contrast to a mode list you can always access one way or another.

This metric scores mode design and offline accessibility specifically — whether enough people are actually playing a given mode right now to get you into a match is a community-scale property, not a mode-design one, and is scored under Community instead. A game can score well here with a thin population, or poorly here despite a healthy one; the two are independent questions.

## Rubric

| Score | Description | What This Means for You |
|---|---|---|
| 5 | Every mode has a distinct identity that complements the maps and class system rather than reskinning the same match with a different win condition. Every mode is playable offline against bots as well as online. At least one mode is memorable enough that the community still talks about it years later. | Whatever mode you're in the mood for, it's there and it plays differently from the others — online with real people, or offline if you'd rather practice or play solo. The mode list isn't padding. |
| 4 | Most modes are well-designed and distinct, with one or two that feel like a lighter variation on another. Most modes have an offline option, even if a niche mode or two lacks one. | You'll have a short list of go-to modes that always deliver and mostly work offline too, with one or two exceptions. |
| 3 | Modes are functional and technically different from each other, but several boil down to the same core loop with a different label. Offline options are limited to a mode or two rather than the full list. | A couple of modes are genuinely worth returning to; the rest feel like the same match wearing a different name, and offline is only a fallback for a couple of them. |
| 2 | Most modes feel like thin variations on a single core mode, with little to distinguish them beyond a rule tweak, and barely interact with the maps or class system. Offline play is missing or an afterthought. | In practice there's really one mode here with everything else theoretical, and nothing to fall back on if you'd rather play solo. |
| 1 | Modes exist on paper but bring nothing distinct to how the maps or class system get used. There's no offline alternative at all. | The mode list is a menu, not a set of ways to actually play, and there's no version of the game you can play on your own. |
