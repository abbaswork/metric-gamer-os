# Metric: Progression
**Genre:** Racing
**Type:** Qualitative
**Direction:** Complaint — 5 = deep, well-structured progression, 1 = no meaningful structure

## Why This Matters

A NeoGAF open letter to racing game producers called out short championship structures as one of the main things that "destroys the experience." The specific complaint: players "barely had time to get into the feel for a particular car class or race type before it was over." The flip side — what players want — is long championships, varied event types, and races where finishing well matters even when you don't win. A separate NeoGAF thread was titled "Why do most racing games have such lackluster single player careers?" — confirming this is a genre-wide pain point, not one game's problem.

For racing games, progression is specifically about how deep and varied the event structure is. It is less about raw playtime and more about whether the game gives you something meaningful to work toward across sessions.

## Sources

- **NeoGAF: "An open letter to racing games producers"** — micro competitions and short championship structures cited as directly destroying the experience; longer races and custom championships explicitly requested (live source, confirmed via web search)
- **NeoGAF: "Why do most racing games have such lackluster single player careers?"** — confirms this is a recurring, genre-wide player frustration (live source, confirmed via web search)
- **r/granturismo** (training data) — GT7 vs GT4 regression is the canonical community example of a strong progression system being gutted; players described it as the main reason GT7 felt like a step backward
- **r/forza** (training data) — community criticism of progression across entries centres on whether unlocks feel earned
- **Steam user reviews** (training data) — "nothing to do after 10 hours" and "just grinding for credits" are consistent patterns in lower-scored racing titles

## Rubric

| Score | Description | What This Means for You |
|---|---|---|
| 5 | Deep championship structure with varied event types and meaningful progression even without winning every race. | You'll keep finding new event types and structures well into your time with the game. Coming second in a chaotic race feels as satisfying as winning cleanly. |
| 4 | Solid championship depth with a range of event types and clear progression. Finishing mid-field carries some value. | You'll have a real sense of building toward something. The structure is substantial enough to get real value from. |
| 3 | Functional championship structure but limited event variety. Mostly short competitions with few event types. | A few hours a week and the structure holds. Heavier sessions will outpace what's there within a few weeks. |
| 2 | Predominantly micro competitions with little variation. Progression feels shallow throughout. | You'll run out of meaningful things to work toward quickly. From that point, the game asks you to repeat what you've already done. |
| 1 | No meaningful championship or progression structure. Single race events with no broader goals. | If you need the game to give you a reason to keep playing, you won't find it here. |
