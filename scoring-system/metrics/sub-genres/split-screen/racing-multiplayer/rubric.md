# Metric: Split Screen Performance
**Sub-genre:** Split Screen Racing
**Type:** Qualitative (community consensus)
**Direction:** Complaint — 5 = no performance degradation, 1 = unplayable in split screen

## Why This Matters

Players and developers both say the same thing about split screen: "There's something about racing against someone physically next to you that is one of those special qualities of racing games." That makes it sting more when performance falls apart. The hardware load of rendering multiple viewports simultaneously is real — developers cite it as the main reason the feature gets dropped — but poor optimisation is widely criticised as an excuse. F1 24 became the community reference point after a patch caused frame rates to drop to what players described as "flip-book" levels during local play. The Forza Motorsport controversy (shipping without split screen entirely, with Microsoft calling it a "creative decision") confirmed how strongly the community values the feature.

## Rubric

| Score | Description | What This Means for You |
|---|---|---|
| 5 | Split screen that holds up against single-player mode. It retains its target frame rate without any visual quality loss. | The difference between two to four players isn't noticeable. |
| 4 | Good performance with the occasional dip under heavy load. Some frame rate variance in the more chaotic moments, but it stays playable throughout. | You might catch a brief stutter in a big pile-up, but it won't affect your racing or make you wish you were playing solo. |
| 3 | Performance is generally stable around 30fps with some reduction of quality compared to solo play. While still playable, the difference from the single-player experience is apparent. | You'll be aware that you're running in a reduced mode. If 60fps solo is what you're used to, the step down is apparent but still doable. |
| 2 | Unstable performance with frequent frame drops and visual degradation. | Performance will be working against you within competitive situations. |
| 1 | Severely degraded performance to the point you choose not to use split screen at all. | The mode exists but it won't take long before you find another game to play. |
