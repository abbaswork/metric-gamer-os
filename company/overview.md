# Metric Gamer — Company Overview


## What We Are

Metric Gamer [https://www.metricgamer.com/] is a tool based game review platform that is gamer-to-gamer.

## Vision

To become the go-to reference to help gamers find games based on consistent metrics for games within a niche. IE. split screen action rpg ps2 games is a niche category that utilzies the same metrics for each game.


## Unique Values Preposition

How do we offer our Unique Value Preposition:

- **Gamer-first** — every score traces back to real player voices
- **Context matters** — metrics are tailored to what each list is actually about
- **No critic bias** — we source from players, not press
- **Genuine value** — we surface real pain points and real highlights, not sanitized takes
- **Community-driven** — the ratings reflect how the community rates a game, not how one person does
- **Scoring system** - each game is scored against a highly researched rubric based on metrics for that specific genre or sub-genre

## How It Works

1. **A campaign is defined** around a specific gaming context or use case (platform, genre, play style, social setting, etc.)
2. **Metrics are assigned** that reflect the core values of that list 
3. **Games are found** that fit into that campaign
4. **Player reviews are researched** for each game across Steam, Reddit, forums, and storefronts
5. **Games are scored** out of 5 based on the rubric of each assigned metric
6. **An introduction, verdict, pros and cons, metric score, and descriptions for each are produced** — grounded in genuine player pain points and praise, not editorial opinion
7. **FAQ is generated** based on common community questions
8. **Ranked Lists are created** that rank games of a shared gaming context by metric scores
