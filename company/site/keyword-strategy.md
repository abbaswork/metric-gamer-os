# Metric Gamer — Keyword Strategy

## The Hierarchy

Keywords drive everything. No campaign structure is proposed before keyword validation is complete.

```
Keyword Research
      ↓
Campaign (one niche — one confirmed modifier combination)
      ↓
Clusters (ranked lists — one per validated Tier 1 keyword)
      ↓
Individual Game Pages (one primary keyword per page — never the bare game name)
```

---

## The DA 2 Reality

At DA 2, Metric Gamer cannot rank for:
- Bare game names ("gran turismo 4", "rocket league") — owned by official sites, Wikipedia, IGN, Metacritic regardless of volume or competition index
- Genre-only queries ("racing games", "football games") — same reason
- Platform-only queries ("PS4 games", "best PS4 games") — same reason
- Any query where IGN, Metacritic, Fandom, or Wikipedia occupy 3+ of the top 5 organic positions

**The traffic model at DA 2 is aggregation, not scale.** No single keyword drives hundreds of clicks per month. Traffic accumulates across many pages each generating 20–100 clicks per month from a long-tail cloud of query variants — not from a single high-volume primary keyword. Strategy optimises for a large number of winnable low-volume positions across a well-structured cluster, not for a few high-volume keywords.

A realistic output from one well-executed campaign cluster (5 pages, all ranking page 1):
- 5 pages × average 30–80 clicks per month = 150–400 clicks per month
- Approximately 35–100 clicks per 28 days from one campaign

500 clicks per 28 days requires multiple campaigns active in parallel, social referral traffic, and backlink-driven DA growth to unlock higher-volume keywords over time. No single campaign delivers it.

## The Modifier Formula

For any list-level keyword to enter consideration, it must carry at least 3 of these 4 modifiers:

- **Genre** — racing, football, FPS, etc.
- **Platform** — PS4, PS5, PC, mobile, etc.
- **Price point** — free, free-to-play, cheap, under £10, etc.
- **Play mode** — multiplayer, 2-player, co-op, split-screen, solo, etc.

| Modifier count | Example | Winnable at DA 2? |
|---|---|---|
| 1 | best racing games | No |
| 2 | best racing games PS4 | Marginal — SERP check required |
| 3 | best free racing games PS4 | Yes — if SERP check passes |
| 4 | best free two player racing games PS4 | Yes — very low competition zone |

As DA grows, one modifier can be dropped per tier of DA gain. At DA 5–7, some 2-modifier queries become viable.

## The Three Keyword Tiers

### Tier 1 — List Intent (Ranked List Pages)
The searcher is choosing between options and hasn't decided yet. These queries typically start with "best" or are phrased as comparisons.

- **Target with:** ranked list pages
- **Minimum modifiers:** 3
- **Placement:** URL slug, H1, meta title, first 100 words, FAQ questions

### Tier 2 — Game-Specific Intent (Individual Game Pages)
The searcher knows the game and is researching or validating a decision. NOT the bare game name.

- Game name + campaign modifier (e.g., "rocket league free ps4")
- Game name + platform + review (e.g., "burnout 3 takedown ps2 review")
- Game name + decision intent (e.g., "is gran turismo 4 worth playing")
- Game name + feature (e.g., "rocket league split screen multiplayer")

**Niche game exception:** For smaller or older titles with minimal major publication coverage, targeting closer to the bare game name is viable — the SERP is likely to show forum threads rather than IGN/Metacritic. Confirm via SERP check.

Game pages are structural. Their primary role is to support ranked list authority and capture Tier 3 long-tail queries. Direct traffic from game page primary keywords will be low (10–100 monthly). This is expected.

### Tier 3 — Question Intent (FAQ Entries)
The searcher has one remaining question before deciding. Lives in FAQ sections on both game pages and ranked lists.

- Often zero volume in the planner but real searches happen
- These are captured as GSC impressions from aggregated long-tail queries
- Examples: "does rocket league have split screen on PS4", "is burnout 3 on PS5", "can you play gran turismo 4 online"

## Keyword Planner vs GSC — What Each Measures

| Tool | What it measures | What it misses |
|---|---|---|
| Google Keyword Planner | Estimated monthly searches for a specific phrase | The long-tail cloud — hundreds of variants each searched a handful of times |
| Google Search Console | Every query that triggered an MG page — actual impressions and clicks | Keywords MG doesn't rank for at all |

A page showing 37,000 impressions in GSC does not mean 37,000 people searched that exact phrase. It means 37,000 combined searches happened across every query variant Google matched to that page. The planner tells you what to target. GSC tells you what's already working. Both are required.

## YoY Trend Filter

Apply before any keyword enters consideration:

- **Positive or +900% YoY:** growing demand — prioritise
- **0% / flat:** stable — acceptable if volume and competition pass
- **-50% to -90% YoY:** structural decline — the platform or topic is losing search interest permanently. Do not anchor a campaign here.

A -90% YoY on PS4-specific queries reflects the PS4 ageing as a platform. This will not reverse. Building a campaign around a structurally declining modifier guarantees diminishing returns.

## The SERP Check

The keyword planner competition index measures paid advertising competition, not organic. A "Low" competition keyword may have IGN and Metacritic on page 1. A "High" competition keyword may have forum threads and small sites ranking organically.

**Before any keyword is confirmed, search it in an incognito window and check:**
1. Who is in positions 1–5 and what type of page (list/review/forum/commercial/video)?
2. Any featured snippets, knowledge panels, or video carousels above organic results?
3. Intent signals — is page 1 directly answering the query, or is the intent split?

**Intent split check:** "best arcade racing games" surfaces physical coin-op machine sellers alongside video game lists. The volume is split across two completely different audiences. Adding a platform or context modifier clears the intent but reduces volume. Always check what Google is serving before committing.

**SERP pass criteria at DA 2:**
- No IGN, Metacritic, Fandom, or Wikipedia in positions 1–3
- At least 2 of positions 1–5 are smaller or medium sites, forum threads, or user-generated content
- Page 1 is directly answering the query
- No video carousels dominating above-the-fold

## Campaign Formulation

A campaign is one modifier combination confirmed by data.

**Confirmed when:**
1. Modifier combination passes the DA 2 filter (3+ modifiers)
2. YoY trend is flat or positive
3. At least one Tier 1 keyword has measurable volume (100+ monthly)
4. SERP check passes
5. Intent is clean — no significant split between different searcher types

**A campaign is not** a genre, a platform, or a content type. "Free arcade racing games" is a campaign. "Racing games" is not.

---

## Cluster Formulation

Each cluster = one ranked list targeting one distinct Tier 1 keyword variant.

**Rules:**
- Maximum one shared game between any two clusters in the same campaign
- Each cluster requires a minimum of 3 supporting game pages already in MG's content library
- All cluster entries must be rewritten from game page source material using the write-ranked-list skill — no direct copy from game pages into lists
- Platform can vary between clusters if game selections are genuinely distinct

## Game Page Keywords — Placement Hierarchy

| Location | Rule |
|---|---|
| URL slug | Primary keyword, exact or close match, max 5 words |
| Meta title | Primary keyword + \| Metric Gamer, under 60 chars |
| Meta description | Primary keyword + one qualifier + CTA, under 155 chars |
| H1 | Same as or very close to meta title, no brand |
| First 100 words | Primary keyword appears naturally |
| H2s | Secondary keywords and topic headings |
| FAQ questions | Tier 3 long-tail variants phrased as real searches |
| Schema | gamePlatform, genre, name — keyword-rich structured data |
| Internal anchor text | Game name + context when linking from list to game page |

Keywords only go where they fit the prose naturally. Forced placement is a negative signal.

---

## Internal Linking

- Every game page links to every ranked list it appears in
- Every ranked list links to all its supporting game pages
- Anchor text: game name + context ("Rocket League's free PS4 multiplayer" not "Rocket League")
- No orphan pages — every game page connects to at least one ranked list
