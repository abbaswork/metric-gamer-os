# Metric Gamer — FAQ Templates

FAQs are tailored to each game but drawn from a defined set of question types grounded in what players actually ask. Every FAQ on a game page should pull from this list based on what is relevant to that game.

**Sources for this document:**
- Fextralife FAQ (Elden Ring) — verbatim community questions
- TheAnalyticalGeek — top 10 most-searched Elden Ring questions
- Nintendo Life — Persona on Switch FAQ (16 questions)
- Windows Central — Shadow of the Erdtree DLC FAQ
- Charlie INTEL, Dexerto, Eldorado.gg — crossplay coverage across multiple titles
- GameFAQs board threads — pre-purchase length and difficulty questions

---

## Universal Questions

These apply to every game. Include all of them unless the answer is not applicable (e.g. a single-platform exclusive removes the crossplay question).

| # | Question Template | What the Answer Must Cover |
|---|---|---|
| 1 | Is [game] cross platform? | Whether crossplay is supported between PS/Xbox/PC. If partial (e.g. PS4/PS5 only), say so explicitly. If absent, state it and why if documented. |
| 2 | What platforms is [game] available on? | Full list of platforms at launch. Note if a platform version came later (e.g. PC port after console). |
| 3 | How long is [game]? | Three numbers: main story only / main + side content / completionist. Source from HowLongToBeat community averages where available. |
| 4 | Is [game] hard? | Community consensus on difficulty. Note if difficulty settings exist. Flag if the game is known for being punishing (Souls-like) or accessible (casual-friendly). |

## Conditional Questions

Include these only when relevant to the specific game.

### Multiplayer / Co-op

| Condition | Question Template | What the Answer Must Cover |
|---|---|---|
| Game has any multiplayer | Does [game] have multiplayer? | Whether online or local. PvP, co-op, or both. Maximum player count. |
| Game has local co-op | Does [game] have couch co-op / split screen? | Whether split screen is supported. How many players. Any limitations (e.g. campaign only, not all modes). |
| Game has online co-op | Can you play [game] with friends online? | Whether co-op is available online. Whether it requires an invitation or matchmaking. |
| Game has crossplay | Does [game] have cross-progression? | Whether save data or purchases carry across platforms. Relevant when players own multiple platforms. |
| Game is primarily multiplayer | Can you play [game] solo? | Whether solo play is viable or if the game is designed around group play. |

### Series / Franchise

| Condition | Question Template | What the Answer Must Cover |
|---|---|---|
| Game is part of a series | Do I need to play the previous games first? | Whether prior entries are required for context. If yes, which ones and in what order. If no, explain what carries over. |
| Series has multiple entry points | Which [series] game should I start with? | A direct recommendation with a one-sentence reason. Do not list all options without a recommendation. |
| Game is a sequel or remake | What is the difference between [game] and [previous version]? | The specific changes: new content, removed content, mechanical differences. Not a marketing summary. |

### DLC / Post-Launch Content

| Condition | Question Template | What the Answer Must Cover |
|---|---|---|
| Game has paid DLC | Does [game] have DLC? | What the DLC adds and whether it is required for the full experience. |
| DLC has a price | How much does the [game] DLC cost? | Price at the time of writing, with a note to verify current pricing. |
| DLC has access requirements | Do I need to finish the game to play the DLC? | Any in-game prerequisites. Whether a new character can jump straight to the DLC. |
| Game has a season pass or editions | What is the difference between the Standard and [other] Edition? | What the premium edition includes and whether the additional content is worth the price difference based on community feedback. |

### Difficulty / Accessibility

| Condition | Question Template | What the Answer Must Cover |
|---|---|---|
| Game is known for high difficulty | Is [game] good for beginners? | Honest answer. If the game is not beginner-friendly, say so. Include any beginner-specific modes or community-recommended starting tips if they exist. |
| Game has no difficulty settings | Does [game] have an easy mode? | Whether the developer offers difficulty options. If not, note any community accessibility workarounds (e.g. summons in Elden Ring). |
| Game is part of a demanding genre | Is [game] like [comparable game]? | A direct answer to the comparison. What carries over from the comparable game and what is different. |

### Platform-Specific

| Condition | Question Template | What the Answer Must Cover |
|---|---|---|
| Game has performance differences across platforms | Does [game] run well on [platform]? | Frame rate, resolution, and any documented performance issues specific to that platform. Source from Digital Foundry or community reports where available. |
| Game is on older hardware | Does [game] run on [last-gen platform]? | Whether the last-gen version exists and any notable differences from the current-gen version. |
| Game is on Game Pass / PS Plus | Is [game] on Game Pass or PS Plus? | Current subscription status. Note that this can change and readers should verify. |

---

## Question Format Rules

- Questions are written in second person ("Is [game] cross platform?" not "Does the game support cross-platform play?")
- Answers are written in second person, direct, and no longer than 3 sentences
- If the answer is "no," say so in the first sentence — do not bury it
- Answers must reflect community-sourced information, not marketing copy
- Do not invent questions for a game that the community does not actually ask

---

## Question Priority Order

When building a FAQ for a specific game, include questions in this order:

1. Crossplay (if multiple platform versions exist)
2. Available platforms
3. Game length
4. Difficulty
5. Multiplayer / co-op (if applicable)
6. Series entry point (if part of a franchise)
7. DLC (if post-launch content exists)
8. Platform performance (if there are documented differences)
9. Game Pass / PS Plus status (if known)
